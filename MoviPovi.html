Project Name
MoviPovi
Type

Technologies
Flutter, Vue, AWS, MySQL, NodeJS,Python

A on-demand movie streaming platform. OTT Platform similar to Netflix but for Local languages. It enables the Local Movie Makers to Post their videos and allows the User to view them upon Subscription.

Features:

  • Stream movie on-demand

  • User can upload their movie clip and sell them to the custom audience

  • Integration of Payment gateway for the users so that they can subscribe to the platform. 2 payment gateways Integrated.

  • Automated Workflow to Split the Subscription payment between the Platform Owners and the Movie Makers.

  • Real time user Statistics with graphs and Charts

  • Push and in-app Notifications Implemented

  • Recommendation System for Movies was Implemented
