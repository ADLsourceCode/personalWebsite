A hyper-local e-commerce marketplace. It offers a shopping experience like never before. You can find stores in multiple product categories such as Grocery, Food, Healthcare and wellness, and many more from store near you. A Total of 3 Apps and 2 Admin Panels were Developed.

Features:

  • 3 Apps – Customer, Seller, Delivery

  • 2 Web Based Admin Panel – Super Admin and Seller Admin Panel

  • Find and explores store near you Based on Location

  • 2 payment gateways Integrated.

  • COD Workflow Implemented

  • Short and Seamless Order Flow, Including Guest Check Out

  • Put Alerts on Product Availability

  • Push and in-app Notifications Implemented

  • Sellers can Keep theirs complete accounts and Transactions

  • Chat with store owner before and after placing order and have a seamless experience.

  • Admin Panel to Maintain and Track Everything

  • Real time user Statistics with graphs and Charts

  • Complete Seller and Admin Accounting System developed under the Admin Panel.

  • Live Reporting about user preferences using User Click Tracking

  • Product Recommendation implemented

  • User Re-targeting System using Admin Panel
